# vue-todo 

做一个简单的vue todo 

在线预览：[https://befriend1314.github.io/vue-todo/](https://befriend1314.github.io/vue-todo/)

后续补充了：

1. css 代码拆分单独打包
2. 业务代码和构建代码区分打包


webpack 版本是 4.29.0 


## 本地开发

```
 # clone 项目
 git clone git@github.com:befriend1314/vue-todo.git

 # 切换目录
 cd ./vue-todo

 # 安装依赖
 npm i


 # 启动服务端
 npm run dev
 
```

浏览器访问控制台输出的地址
